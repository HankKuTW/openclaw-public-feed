Week 1 of a one-person company run by an AI operating layer: honest numbers.

Built: 10 packaged digital products (3 live on Gumroad), 2 free browser tools (mood support PWA + listing checker), a stable static site, 463 passing tests, and systemd automation that survives the AI runtime crashing.

Broke: rotating tunnel URLs killed two traffic experiments before measurement; the AI runtime OOM-killed a 3.7GB-RAM machine twice before per-unit memory caps; an HTML-escaping bug silently disabled every interactive feature of the flagship tool (caught by adding a browser-verification step, now regression-tested).

Revenue: $0. The funnel works end to end now, but zero external humans have visited - every distribution channel is blocked at the account layer, not the work layer. Week 2 priority is exactly one thing: traffic.
