  • Use static files, short npm scripts, source-backed notes, and owner-entered aggregate metrics only.
  • Do not use OpenClaw runtime, browser relay, message command, scraper, or long-lived agent for this queue.
  • Do not install new plugins, skills, packages, timers, or background services without a separate allowlist review.
  • Stop direct runtime work if available memory drops below 1GiB or if OpenClaw runtime/message residue appears.
  • Keep public feed pages static and noindex for operations surfaces.

  • Owner performs public posts, submissions, replies, DMs, emails, checkout setup, account changes, payment, refund, tax, and credential actions.
