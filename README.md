# OpenClaw Public Feed

Free, local-first browser tools from a one-person digital product studio.
Everything runs in your browser; no accounts, no server-side data collection.

**Live site: https://hankkutw.github.io/openclaw-public-feed/**

## Free tools

| Tool | What it does |
| --- | --- |
| [Mood Anchor (English)](https://hankkutw.github.io/openclaw-public-feed/mood-anchor-en.html) | Offline-first mood support PWA: check in, get one tiny next step, build a 7-day micro-action plan. Data stays in your browser. Not a medical product. |
| [Mood Anchor（繁體中文）](https://hankkutw.github.io/openclaw-public-feed/mood-anchor.html) | 離線可用的情緒支援小工具；資料只存在你的瀏覽器。非醫療產品。 |
| [Self-Care Pocket Card](https://hankkutw.github.io/openclaw-public-feed/mood-anchor-care-card-en.html) | One printable page: tiny actions by energy level, crisis resources, and a trusted-contact line. |
| [Listing Friction Grader](https://hankkutw.github.io/openclaw-public-feed/listing-friction-grader.html) | Paste a digital product listing and get a local-only friction score plus the next three fixes. |

## Privacy stance

- All tools work entirely in the browser (localStorage only).
- No analytics scripts, no cookies, no accounts.
- Mood Anchor is not diagnosis, treatment, or a crisis service; it links to
  real crisis resources (988 and local emergency services) first.

## About

Built and operated by a one-person company experiment: an AI operating layer
(OpenClaw) handles internal production while the owner keeps all external
authority. This repository is the static build of the public site.
