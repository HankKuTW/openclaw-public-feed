  • codex_review_after_openclaw_packet: After OpenClaw produces or updates a packet, Codex reviews evidence, safety wording, and low-load constraints before publishing.
  • owner_escalation_only_for_external_authority: Escalate to the owner only for login, public posts, replies, DMs, emails, checkout, payment, account settings, credentials, or real-world commitments.
  • oom_guard_before_runtime_expansion: Do not expand runtime or background services; stop direct runtime work if available memory drops below 1GiB.
  • telegram_pending_until_credentials: Telegram notification remains pending until direct Bot API token and chat id are configured securely.
